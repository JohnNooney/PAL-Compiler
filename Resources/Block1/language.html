The Block-1 Language

The Block-1 language is a very simple block-structured language. It is free-format but case-sensitive both for keywords and variable names. A Block-1 program is zero or more variable declarations followed by either a single statement or a begin...end block, the block can also be used as the body of a for statement.

The scope of all variable names is the entire program. Only integer and real types are used in the language and it is strictly typed with no type coercion; i.e. you cannot mix integers and reals.

Variables can be assigned to using a let statement. The usual form of an arithmetic expression is available with addition, subtraction, multiplication and division operators.

A for statement provides a looping mechanism.

An Extended BNF Specification

Block-1.EBNF.txt is an extended BNF (EBNF) specification of the Block-1 language.

Note that this extended BNF specification defines exactly the same language as the standard BNF specification. That is exactly the same programs in the language are valid and that the same semantics apply.

This is an example of the relationship between a language and its BNF specification; i.e. one language may be defined syntactically by many equivalent BNF and/or EBNF specifications.
