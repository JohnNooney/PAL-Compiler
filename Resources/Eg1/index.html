The Eg1 Case Study

This case study illustrates the processing of some tokens for a basic language using an FSM as a scanner implementation.

The tokens processed are:

Identifier   Integer   =   >   >=

A C++ scanner is implemented and demonstrated through a C++ application. The implementation of the scanner to process the above tokens is built on the CompilerKit recursive-descent compiler toolkit. The token and Scanner classes of the toolkit are used as the basis for this implmentation.

The following files make up the case study:

  • Eg1.FSM.txt: an FSM for the tokens in text format.
  • Eg1Scanner.hpp/.cpp: the scanner class for finding Eg1 tokens. This is inherited from the toolkit Scanner abstract class, implementing the lex() protected abstract method using an FSM.
  • test1.txt, test2.txt, and test3.txt: example input files used for testing.
  • makefile: a compile script to be used with make to compile the application into an executable. alternatively, you import the source files into your usual IDE.
